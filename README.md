# corona-ehime-data


## 目的
愛媛県の新型コロナのデータをjson形式でほしかったため、変換したものを公開しています。


## データ元
[愛媛県のオープンデータのサイト](https://www.pref.ehime.jp/opendata-catalog/)のうち、
[新型コロナウイルスの検査状況等について](https://www.pref.ehime.jp/opendata-catalog/dataset/2174.html)
についてのデータを使用しています。
